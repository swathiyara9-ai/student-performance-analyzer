# Student Performance Analyzer

A Python program to analyze student marks and visualize performance using graphs.

## Features

- Add and view student records
- Calculate class average
- Identify topper
- Grade classification
- Grade statistics
- Data visualization using graphs
- Save and load data using CSV files

## Technologies Used

- Python
- Matplotlib
- CSV file handling

## How to Run

1. Install matplotlib
   pip install matplotlib

2. Run the program
   python student_analyzer.py
